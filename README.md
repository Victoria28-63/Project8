# Currency-Converter
## О проекте
Desktop-приложение для конвертации валют с использованием актуальных курсов из внешнего API. Программа сохраняет историю конвертаций в JSON-файл и позволяет загружать, очищать и экспортировать историю.

## Возможности

- Получение актуальных курсов валют через API
- Поддержка всех мировых валют
- Проверка корректности ввода (только положительные числа)
- Автоматическое сохранение истории в JSON
- Загрузка истории при запуске
- Очистка и экспорт истории
- Интуитивно понятный GUI на Tkinter
